# What this module covers

Every session with Boomi Companion should end with a short, deliberate stop. Not a "close the tab" stop. A reflection that captures what worked and what didn't, so the next session is faster and more accurate.

# What to capture: friction

Anything that provided friction this session. Specifically:

- A misunderstanding that took a redirect.
- A wrong assumption Boomi Companion made.
- A way of phrasing something that landed better than your first try.

The next session will be faster if Companion knows what this one learned.

# What to capture: wins

Anything Boomi Companion did particularly well:

- A clean pattern.
- A structure you want to reuse.
- A default that turned out to be right.

These become starting points you can ask Companion to follow next time.

# Why this matters

Without the stop, every session starts cold. With the stop, sessions compound — each one leaves behind a small residue of hard-won context that the next one can pick up.
