# Overview

*Co-Creating with Boomi Companion* is a Boomi L&D course that teaches integration developers how to work alongside [Boomi Companion](/concepts/boomi-companion.md), Boomi's AI development partner. The course frames the developer's role as a co-creator rather than a passive prompter: bringing clear scope, asking senior-developer questions before opening the tool, reviewing what Companion produces, and closing each session with a deliberate reflection so the next session runs faster.

# Learning intent

By the end of the course, a developer should be able to:

1. Explain what Boomi Companion is and how it fits into the Boomi platform.
2. Distinguish between the [conversational](/concepts/conversational-mode.md) and [canvas](/concepts/canvas-mode.md) ways of using Companion.
3. Scope a new integration request with the questions a senior developer would ask before opening the tool. See [Planning Together](/modules/04-planning-together.md).
4. Review Companion's output on the canvas, including map steps, groovy scripts, and test executions. See [Developer Build Walkthrough](/scenarios/developer-build-walkthrough.md).
5. Close a Companion session with a reflection on friction and wins so future sessions build on what was learned. See [Stopping](/modules/05-stopping.md).
6. Build habits for [staying current](/modules/06-habits-for-staying-current.md) as Companion and the platform evolve.

# Structure

The course is organized into six modules:

1. [Introduction](/modules/01-introduction.md) - what Boomi Companion is and how it works
2. [Conversational and Canvas](/modules/02-conversational-and-canvas.md) - the two ways to work with Companion
3. [Connector Registry](/modules/03-connector-registry.md) - how Companion discovers what it can build with
4. [Planning Together](/modules/04-planning-together.md) - scoping a request before you prompt
5. [Stopping](/modules/05-stopping.md) - the deliberate close-out of a session
6. [Habits for Staying Current](/modules/06-habits-for-staying-current.md) - keeping pace as the tools evolve

# Audience

Boomi integration developers who are new to working with Boomi Companion, or who have used it casually and want to work with it more deliberately.
